# 🧠 Deep Learning Projects Portfolio

This repository contains a collection of **Deep Learning projects** built using Python and modern AI frameworks such as **TensorFlow, Keras, and Scikit-learn**.  
The projects demonstrate practical applications of **Neural Networks, Convolutional Neural Networks (CNNs), and optimization techniques** for solving real-world problems in computer vision and predictive analytics.

Each project covers the **complete machine learning workflow**, including data preprocessing, model building, training, evaluation, and in some cases deployment using modern web technologies.

## 🥔 Potato Disease Classification using CNN

### Tech Stack
Python, TensorFlow/Keras, FastAPI, React, Docker, TensorFlow Serving, Deep Learning (CNN)

### Description
An end-to-end deep learning application that detects diseases in potato plant leaves using **Convolutional Neural Networks (CNN)**.  
The model is trained on image data to classify potato leaves into **Healthy, Early Blight, and Late Blight** categories.

The system includes a **complete deployment pipeline** where the trained model is served using **TensorFlow Serving**, exposed through a **FastAPI backend**, and connected to a **React frontend** for real-time predictions.

This project demonstrates how **AI can assist farmers in early disease detection**, helping reduce crop loss and improve agricultural productivity.

🔗 Repository:  
https://github.com/srinitish/potato-disease-classification-cnn

## 🏠 House Price Prediction using Neural Networks and SGD

### Tech Stack
Python, TensorFlow/Keras, NumPy, Pandas, Scikit-learn

### Description
A deep learning-based **regression model** built using **Keras** to predict house prices based on features such as **area and number of bedrooms**.

The project focuses on implementing **Stochastic Gradient Descent (SGD)** as the optimization algorithm and explores how gradient descent techniques influence training performance.

Key steps include:

- Data preprocessing and cleaning  
- Feature scaling and normalization  
- Neural network model training  
- Performance evaluation using **MSE and MAE**

🔗 Repository:  
https://github.com/srinitish/House-Price-Prediction-using-Stochastic-Gradient-Descent-SGD-/tree/main

## ✍️ Handwritten Digit Classification using ANN & CNN (MNIST)

### Tech Stack
Python, TensorFlow/Keras, NumPy, Matplotlib, Deep Learning (ANN & CNN)

### Description
A deep learning project that classifies handwritten digits **(0–9)** using both **Artificial Neural Networks (ANN)** and **Convolutional Neural Networks (CNN)**.

The models are trained on the **MNIST dataset**, which contains **70,000 grayscale images of handwritten digits** with a resolution of **28×28 pixels**.

This project compares the performance of **ANN and CNN architectures** for image classification tasks.

CNN achieves better accuracy by automatically learning **spatial features from images**, which traditional fully connected neural networks cannot efficiently capture.

The project also visualizes predictions, training accuracy, and loss metrics.

🔗 Repository:  
https://github.com/srinitish/Hand-written-digit-classification-MNIST-with-both-ANN-and-CNN
